# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, constructive, and welcoming. Harassment and abusive behaviour are
not tolerated. Report concerns to the maintainer privately via
[GitHub](https://github.com/Kuzmenko-Pavel).
